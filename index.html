GENeRAL information

  • Any uninitialized variable has the Integer value 0.
  • You can’t access variables that are in another “function” (but you can access other functions except if you redeclared them in your local function)
  • A long press on any function or method key will show you its prototype (ie long press on insert will display "insert(Integer, Any)")
  • If no return statement is used, the value returned by the last line of the script is used as return value.

METHODS

  • Methods can be called on Lists. (The only exception is “is_list” which can be called on any type). Syntax: list.method(parameters).  Parentheses may be omitted if no parameters are being provided, ex List.pop instead of List.pop().
  • [List].length() Integer
    returns the number of elements within the list

  • [List].push(value Any) List
    adds a value to the end of the list and returns the list.

  • [List].pop() Any
    returns the last value from the list and removes it.

  • [List].insert(index Integer, value Any)
    inserts the value at the given index of a List

  • [List].remove(index Integer)
    removes the value at the given index from the list. List index starts with 0

  • [List].sort()
    sorts in place the list ascending 

  • [List].sort_with(func function(a Any, b Any) Boolean)
    sorts in place a list by calling the given function to compare the values

  • [List].map(func function(value Any) Any)
    applies the given function to each entry of the list.

  • [List].copy() List
    creates and returns a copy of the list. Nested list won’t be copied.

  • [Any].is_list() Boolean
    returns true if the sender is a List.

  • [List].fill(value Any) 
    Sets every element of the list to the given value

CUSTOM FUNCTIONS
You can define up to 16 custom numbered functions.
Functions can be stored to variables and lists and can also be redefined later. So if you need more than 16 functions you can either nest them or store them in a list.
All parameters are passed as value beside of Lists which are passed as reference. So if you modify the list inside of a function, the original list will be modified too!

Example:

function f1: var_a, var_b {        // Function F1 takes two parameters (var_a, var_b)
        return var_a * var_b;     // Return the result of a * b
}

﻿Anonymous Functions﻿
You can define brief anonymous functions (only one liners).

Example:

function var_a, var_b -> var_a + var_b

The main purpose is to use it to do one liners like:

var_a.sort_with(function var_a, var_b -> var_b > var_a);    // reverse sort
